# Game Library

A modern React-based web application that showcases a collection of popular games with detailed information, ratings, and visuals. The project allows users to explore games, view their details, and experience smooth animations and authentication features powered by Firebase.

## Live Demo

**Live URL:** [Visit Game Library](https://classy-praline-339ca0.netlify.app/)

## Project Purpose

The **Game Library** website is designed to provide users with an interactive platform to:
- Browse popular games in an organized layout.
- View individual game details, ratings, and descriptions.n
- Log in using email/password or Google authentication.
- Experience smooth UI transitions and animations.

The goal of this project is to demonstrate skills in **React.js**, **Firebase Authentication**, **Context API**, and **UI frameworks** such as **Tailwind CSS** and **DaisyUI**.

## Key Features

- **Authentication System** — Firebase-based login and registration with email/password & Google.
- **Dynamic Game Cards** — Data loaded from JSON and displayed as interactive cards.
- **Sorted by Ratings** — Games are shown in descending order based on their ratings.
- **Private Routes** — Access to certain pages restricted to logged-in users only.``
- **Animated Banner** — Smooth, motion-based transitions using Framer Motion.
- **Responsive UI** — Fully mobile-friendly layout using Tailwind CSS and DaisyUI.
- **Environment Variables** — Secure handling of Firebase configuration using .env file.
